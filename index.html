About the Project

The Personal Expense Tracker is a desktop application designed to help users manage their finances by tracking income, expenses, account balances, and transfers.

Planned Features

  • User registration and login
  • Profile management
  • Create and manage accounts
  • Forgot Password
  • Create Group
  • Add income and expense transactions
  • View account balances
  • Upload receipts, Scan receipts etc
  • Admin dashboard for managing users
  • New Recurring
  • Charts, reports, and PDF export

Optional Features

  • Multi Factor Authentication
  • AI Assistant

Technology Stack

  • Frontend: HTML, CSS, JavaScript
  • Backend: C#
  • Database: To be decided
  • Security: Password hashing and role-based access

Project Timeline

Week 1: Planning, requirements, wireframes, database design

Week 2: User login, registration, and profile features

Week 3: Transactions, balances, and transfer features

Week 4: Testing, debugging, and final improvements
